# Marketing Website

Next.js, TypeScript, and Tailwind CSS marketing website for the Adaptive Smart Irrigation and Crop Optimization Platform.

## Criteria Covered

- Home
- Domain
- Milestones with a selectable assessment menu
- Documents
- Presentation slides
- About us
- Contact us

The app is configured with `output: "export"` so it can produce static HTML under `out/` for course web hosting.

## Run

```bash
npm install
npm run dev
```

## Build Static Site

```bash
npm run build
```

Submission PDFs and slide decks can be placed under:

- `public/submissions/documents/`
- `public/submissions/slides/`
